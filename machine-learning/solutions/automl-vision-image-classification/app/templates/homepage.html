Google Cloud Platform - AutoML Vision API - Sport Classification

This Python Flask application demonstrates App Engine Flexible, Google Cloud Storage, Datastore, and the AutoML Custom Vision API. It is trained to label photos from 10 popular sports in the Summer Olympic Games. The categories are Athletics, Baseball, Basketball, Cricket, Golf, Gymnastics, Soccer, Swimming, Tennis, and Volleyball. The model was trained by uploading 100 images from each category.
